> - This is my default neovim config.

## Setup
> - Run the following commands below.
```bash
cd
cd .config/nvim
touch init.vim
```
> - Copy the `init.vim` content.
> - To locate the `coc.nvim` cd to the following below.
```bash
cd ~/.local/share/nvim/plugged/coc.nvim
yarn install
yarn build
```
